# How to Delete LN-Private/utsav Repository

## Steps to delete the repository:

1. **Go to**: https://github.com/LN-Private/utsav

2. **Click Settings** (gear icon ⚙️ at top right)

3. **Scroll to bottom** → "Danger Zone" section

4. **Click "Delete this repository"**

5. **Type** `LN-Private/utsav` to confirm

6. **Click** "I understand the consequences, delete this repository"

## Note:
You must be an owner/admin of the LN-Private organization to delete it. If you don't have access, ask the organization owner to delete it.
